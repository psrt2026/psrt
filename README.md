# P字幕

一个精简的 Chrome 扩展，只做两件事：

- 加载本地 `.srt` 字幕文件
- 在当前网页的 HTML5 `video` 元素上同步显示字幕

## 功能范围

- 支持在插件弹窗中选择或拖拽 `.srt` 文件
- 使用 `chrome.storage.local` 保存当前字幕内容和开关状态
- 自动检测当前页面最主要的视频元素，并在视频区域底部显示字幕

不包含以下能力：

- 登录
- 下载
- 更新检查
- 远程接口请求
- 站点特化逻辑

## 安装

1. 下载本项目
2. 浏览器打开 `chrome://extensions/`
3. 开启“开发者模式”
4. 选择“加载已解压的扩展程序”
5. 选择当前项目目录

## 使用

1. 打开包含 HTML5 视频的网页
2. 点击扩展图标
3. 导入本地 `.srt` 文件
4. 播放视频，字幕会显示在视频底部
